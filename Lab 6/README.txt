Description:
	1. A 3D scene with Unity built-in cloth component applied.
	2. A controlable and animated player character.
	3. A flag created by Blender with Blender built-in cloth feature, and animated. 
	4. Damageable trees.
	5. Construction system: Player is able to farm trees to collect woods and build constructions. 
	6. UI. 

Control:
	WASD: Move.
	Shift: Run.
	Left Mouse Button: Attack / Construct
	Mouse Move: Control view camera rotation.
	P: Open construction UI
